# pros4noobs-group-project-guidelines
Guia oficial da Pros4Noobs para criadores de Projetos em Grupo, com diretrizes, templates e exemplos de como escrever boas especificações. O objetivo é garantir projetos bem definidos, avaliáveis coletivamente e alinhados com a visão da plataforma, promovendo aprendizado colaborativo, desenvolvimento de portfólio e geração justa de Sparks e Flares.
